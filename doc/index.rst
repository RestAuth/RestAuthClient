Welcome to RestAuthClient's documentation!
==========================================

*RestAuthClient* is a client library for the `RestAuth protocol 
<https://restauth.net>`_ and is the reference client implementation.

This page serves as the API documentation for the library. For further information, please see:

* Official webpage: `<https://redmine.fsinf.at/projects/restauth-python>`_

  * Wiki: `<https://redmine.fsinf.at/projects/restauth-python/wiki>`_
  * Bugtracker: `<https://redmine.fsinf.at/projects/restauth-python/issues>`_
  * Downloads: `<https://redmine.fsinf.at/projects/restauth-python/documents>`_

* git repository: `<http://git.fsinf.at/restauth/python>`_
